2048-angular

2048 game interpretation in AngularJS

View on GitHub Download .zip Download .tar.gz

My interpretation of famous 2048 game in AngularJS framework.

This game is based on online tutorial, that was published originally on ng newsletter.

Build with current Front-end technology.

To make thist game work i used several technology e.g.- Yeoman, Grunt, Karma and SASS.

Simple Backend with Parse

I store data about current high scores inside Parse cloud, so i don't need my own server with database to do this.

How to install 2048-angular game ?

To check how this game work in real, you need to download repository. Then open your command line, move to repository location and hit:

  • npm install
  • bower install
  • grunt serve
